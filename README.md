Uliweb-redbreast 文档与示例
===========================

关于 Redbreast (知更鸟)
----------------
Uliweb是一个Python的web框架，由limodou <limodou@gmail.com> 创立。
Redbreast是一个用于Uliweb的app集合，提供简单的工作流引擎。

本仓库提供中文的redbreast使用说明及示例程序

示例程序
-------
* BatchDemo 
    一个内存工作流使用示例

* YesNoDemo
    Uliweb项目，简单的同意与拒绝流程

* ApproveDemo
    一个完整的Uliweb项目，定义了一个审批流程，有分支结点。
    流程支持意见填写，分支选择，会记录所有的流向日志。
    支持同步或者异常方式来处理流程运转。
